1 # NADCON5-ng
2 
3 Tweaks and Updates to US National Geodetic Survey `NADCON5` Tool. Used to convert Geodetic Data between various US Datums, including: US Standard Datum (`USSD`) used prior to `NAD27`, North American Datum of 1927 (`NAD27`), and various realizations of the North American Datum of 1983 `NAD83`
4 
5 [Link To Doxygen Documentation Website](http://docs.nc5ng.org/latest)
6 
7 The intent of this fork is to adapt the existing tool to be accessible to more users, developers, and data scientists, through the implementation of additional interfaces and workflows on top of existing NADCON5 Code.

47 ### What is NADCON 5.0?
48 
49 NADCON 5.0 performs three-dimensional (latitude, longitude, ellipsoid height) coordinate transformations for a wide range of datums and regions in the National Spatial Reference System. NADCON 5.0 is the replacement for all previous versions of the following tools:
50 
51 - NADCON, which transformed coordinates between the North American Datum of 1927 (NAD 27) and early realizations of the North American Datum of 1983 (NAD 83), and
52 - GEOCON, which transformed coordinates between various latter realizations of NAD 83.
53 
54 ### How do I use NADCON 5.0?
55 
56 NADCON 5.0 is functionally implemented in NGSâ€™s Coordinate Conversion and Transformation Tool. Unlike earlier versions of NADCON and GEOCON, NADCON 5.0 is not a stand-alone tool.
57 
58 Visit the NADCON 5.0 Digital Archive to access raw transformation data that make up NADCON 5.0 (e.g., grids, images, software).
59 
60 ### How can I learn more about NADCON 5.0?
61 [NOAA Technical Report NOS NGS 63 (PDF, 17 MB)](https://www.ngs.noaa.gov/PUBS_LIB/NOAA_TR_NOS_NGS_63.pdf) provides detailed information on NADCON 5.0, and the digital archive includes plots and data.

88 ### Build Options
89 
90 The configurable options for the build steps are
91 
92 1. `OLD_DATUM` - source datum (default: `ussd`)
93 2. `NEW_DATUM` - target datum (default: `nad27`)
94 3. `REGION` - geographical region (default: `conus`)
95 4. `GRIDSPACING` - Grid Spacing in arc-seconds (default: `900`)
96 5. `MAP_LEVEL` - Map Resolution Flag (default: `0`)
97 
98 These can be set as environment variables or directly on the command line
99 
100  export OLD_DATUM=nad27
101  export NEW_DATUM=nad83
102  make
103  # Equivalent
104  OLD_DATUM=nad27 NEW_DATAUM=nad83 make
